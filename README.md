# 🏎️ Dashboard de Análisis del Mercado Automotriz

Un pipeline completo de ciencia de datos enfocado en el mercado de vehículos usados, abarcando desde la limpieza inicial del dataset hasta la creación de un tablero analítico e interactivo en vivo. 

🔗 **[Ver la Aplicación en Vivo aquí](https://fg8r67tiijufajvnnktknb.streamlit.app/)**

---

## 📊 Descripción del Proyecto

Este proyecto analiza las tendencias comerciales, depreciación de marcas y comportamiento del mercado automotriz. Se diseñó un flujo eficiente en Python que procesa datos crudos extraídos de transacciones comerciales, estructurando variables clave (como el año de venta, la marca, el modelo y el precio sugerido) para descubrir qué factores impactan realmente en el valor de reventa.

A partir de los datos procesados, se construyó una interfaz interactiva en la nube que permite a los usuarios segmentar el mercado dinámicamente.

## 🛠️ Tecnologías Utilizadas

*   **Python 3.14**: Lenguaje principal del desarrollo.
*   **Pandas & NumPy**: Manipulación, limpieza de datos, manejo de nulos y estructuración del data pipeline.
*   **Streamlit**: Framework ágil utilizado para el despliegue de la interfaz web.
*   **Plotly Express**: Generación de gráficos dinámicos e interactivos (análisis de tendencias, distribución de precios y volúmenes de venta).
*   **Git**: Control de versiones del código fuente.

---

## 📈 Características del Dashboard

1. **Métricas de Rendimiento (KPIs):** Monitoreo en tiempo real del precio promedio de venta, volumen total del mercado analizado y valoraciones de estado mecánico.
2. **Filtrado Dinámico:** Segmentación por marcas comerciales (`make`) y años específicos de transacción para aislar comportamientos puntuales de mercado.
3. **Explorador del Registro:** Acceso directo a las últimas transacciones procesadas con soporte visual dinámico de los modelos consultados.
